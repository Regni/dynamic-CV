Projects

Rock,Paper,Scissors

A little rock, paper, scissors game where you play against the computer. The score keeps track of how many games you win, lose or draw. If you want to reset the score, just click on the reset button and the scoreboard will be reset, ready for you to win again.

Random Number Game!

A little random number game where you have 5 chances to pick a number between 1 and 100. The game will give you feedback if your choice was too high or too low. If after 5 chances you have not found the correct number, the game will tell you the correct number and you will have to click the reset button to start again.

Affixes this week

A small website created to show what affixes are in World of Warcraft (WOW) this week. Affixes are a way to make some of the gameplay elements in WOW harder, and they are replaced with a new set every week. This website uses the raider.io API to retrieve this information and display it in a simple css grid map.
